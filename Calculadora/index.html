🔢 Decimal para Binário!

Você sabia?
Um número decimal é como contamos normalmente (0, 1, 2, 3...).
Um número binário só usa os números 0 e 1.

Vamos aprender a transformar um número decimal em binário, passo a passo!

📘 Como funciona?

  • Vamos dividir o número por 2 várias vezes.
  • Vamos guardar o resto (0 ou 1) de cada divisão.
  • Depois, vamos juntar os restos de trás para frente.

🎉 Pronto! Isso é o número em binário!
